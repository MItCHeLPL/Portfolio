Description

Create Simple or Perlin Noise and save it as .PNG file.
Created using HTML Canvas and custom JavaScript.

Features

  • Choose Resolution
  • Choose scale
  • Choose background color
  • Choose amount of layers for Perlin Noise generation
  • Choose color channels to be included in generated noise
  • Save as .PNG file
